                                     	          == 만들고 싶은 게임 ==
1.RPG게임 - 카툰형 그림체의 게임 -  MARSHMALLOW GO! 라는 게임.
스토리 - 마시멜로 캐릭터들이 사탕마왕을 막는 게임.
게임 방식 - 1.튜토리얼을 진행한다.
2. 직업 - 5LV를 달성한 후 전직소로 가서 전직 퀘스트를 받는다.
3. 주민들을 만나면서 서브 퀘스트를 받는다.
4. 이런식으로 스토리를 진행하다가 만랩 달성 또는 메인 퀘스트의 마지막 퀘스트를 통과하면 마왕전을 치르게 된다.(만렙은 150LV이다.)

- 이스터 에그 - 튜토리얼 마왕전을 클리어하면 게임의 난이도가 급상승한다.
- 2차 전직소에서 =기타=를 클릭한 후 해골문양을 택하면 네크로멘서로 전직할 수 있다.
- 전직을 하지 않고 60LV까지 갔을 경우 직업이 '올라운더'로 결정된다.

- 기타 - 아이템은 사냥을 하거나 퀘스트 또는 재료를 모아 직접 만들수 있다.
- 무기형 아이템의 종류는 검,방패,방패검,활,도끼,망치,대검,석궁으로 나눠진다.
- 아이템의 희소성은 일반,레어,울트라 레어,에픽,유니크,레전드로 나눠진다.(특별 아이템은 등급이 나눠지지 않는다.)(특별 아이템은 특수 퀘스트를 하여 성공하면 얻을수있다.)
- 스킬은 직업에 따라 다르고 특수 스킬은 2차 전직을 하면 얻을 수 있다.
- 검사 : 기본공격 : 베기 , 3번째 기본공격은 찌르기.
스킬 : 1.내려베기 : 검을 들고 아레로 휘둘러 범위 안에 있는 적들에게 데미지를 준다.
2.돌진 : 검을 들고 돌진을 하여 적에게 데미지를 준다.
3.연속베기 : 검을 3번 앞으로 휘둘러 적에게 데미지를 준다.(만약 무기가 대검일시 스턴.)
4.반격 : 최근에 피해를 입힌 적에게 똑같은 데미지를 주고 적을 밀친다.
